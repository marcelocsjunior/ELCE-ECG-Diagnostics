# 10_GITHUB_BOOTSTRAP

## Objetivo

Criar o repositório GitHub sem perder aderência à baseline real.

## Nome sugerido do repositório

`ELCE-ECG-Diagnostics`

## Visibilidade sugerida

Privado, nesta fase.

## Descrição sugerida

Ferramenta read-only em PowerShell 5.1 para diagnóstico operacional de ambiente ECG com laudo HTML como entrega principal.

## Passos recomendados

1. Criar o repositório vazio no GitHub
2. Não adicionar README automático do GitHub
3. Subir este starter pack como primeiro commit
4. Criar tag `v1.0.0`
5. Publicar release `baseline operacional governada`

# 09_FAQ_TECNICO

## Qual é a porta principal da ferramenta?
`ELCE_ECG_Diagnostics_Menu.bat`

## Onde fica o core?
`C:\ECG\Tool\ELCE_ECG_Diagnostics.ps1`

## Qual é a principal entrega?
`ELCE_ECG_Diagnostics_Report.html`

## Onde ficam os artefatos?
- `C:\ECG\Output\Runs\<RunId>`
- `C:\ECG\Output\Latest`

## A ferramenta corrige algo automaticamente?
Não. O modo atual é read-only.

## Posso refatorar o core livremente?
Não é recomendado nesta fase. A baseline deve ser preservada.

## Posso reativar GUI ou instalador?
Não como prioridade. Esses componentes estão congelados.

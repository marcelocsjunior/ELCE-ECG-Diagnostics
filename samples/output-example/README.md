# output-example

Esta pasta deve conter **artefatos sanitizados** de uma rodada real.

## Objetivo
- demonstrar a estrutura de saída;
- apoiar documentação;
- facilitar onboarding e validação manual.

## Arquivos esperados
- `execution.log`
- `context.json`
- `benchmark.json`
- `timeline.json`
- `analysis.json`
- `ELCE_ECG_Diagnostics_Report.html`
- `ELCE_ECG_Diagnostics_Summary.txt`
- `ELCE_ECG_Diagnostics_Summary.json`

## Observação
Não versionar saídas reais de produção com dados sensíveis.
